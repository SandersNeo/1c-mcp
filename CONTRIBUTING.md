# Как добавить проект в каталог

Спасибо за интерес! Ниже — правила и шаблон для добавления нового MCP-сервера.

## Требования

- Проект должен быть связан с 1С:Предприятие и реализовывать или использовать MCP (Model Context Protocol)
- Для open-source: публичный репозиторий с README и лицензией
- Для коммерческих: рабочий сайт с описанием продукта

## Шаблон записи

Каждая запись в каталоге должна содержать:

```markdown
### [Название](https://github.com/owner/repo)

Краткое описание (1–2 предложения): что делает, для кого, чем уникален.

| | |
|---|---|
| **Язык** | Go / Python / TypeScript / ... |
| **Транспорт** | stdio / SSE / Streamable HTTP |
| **Требования** | 1С:Предприятие 8.3.x+, Docker, ... |
| **Статус** | ✅ Active / 🚧 Beta / 🔬 Experimental / 📦 Archived |

**Возможности:**
- Фича 1
- Фича 2
- Фича 3
```

### Обязательные поля

| Поле | Описание |
|------|----------|
| Название + ссылка | Имя проекта и URL (GitHub или сайт) |
| Описание | 1–2 предложения о назначении |
| Язык | Основной язык реализации |
| Транспорт | MCP transport: `stdio`, `SSE`, `Streamable HTTP` |
| Требования | Минимальные зависимости для запуска |
| Статус | Зрелость проекта (см. ниже) |
| Возможности | 3–7 ключевых фич |

### Статусы проектов

| Статус | Значение |
|--------|----------|
| ✅ Active | Активная разработка, стабильная версия |
| 🚧 Beta | Работает, но API может измениться |
| 🔬 Experimental | Ранний прототип, не для продакшена |
| 📦 Archived | Не поддерживается, код доступен |

## Куда добавлять

1. Запись в соответствующую категорию в `README.md`
2. Строку в сводную таблицу (Open Source или Коммерческие)
3. Строку в CHANGELOG.md

## Процесс

1. **Issue** — если хотите предложить, но не хотите оформлять PR
2. **Pull Request** — если готовы добавить запись сами:
   - Форкните репозиторий
   - Добавьте запись по шаблону выше
   - Откройте PR с заполненным описанием

## Правила

- Один PR = один сервер (или группа от одного автора)
- Описание на русском языке
- Не добавляйте проекты без рабочего кода или документации
- Коммерческие продукты — в раздел «Коммерческие продукты» с указанием цены
